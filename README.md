# dotfiles

My personal configurations for Linux workstations

### Install

First, clone the repository to your computer under `~/.dotfiles` folder, then

```bash
cd ~/.dotfiles
./install
```

### Uninstall

```bash
cd ~/.dotfiles
./uninstall
```
